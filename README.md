# Nova AI Assistant

Nova AI Assistant is an advanced Python-based virtual voice assistant designed to automate desktop tasks, web operations, and voice-controlled interactions. The assistant uses speech recognition and text-to-speech technologies to provide a smart and interactive user experience.

---

## Features

- Voice command recognition
- Text-to-speech interaction
- Open and control websites
- YouTube automation
- Google search automation
- Instagram operations
- WhatsApp Web integration
- GitHub and StackOverflow access
- Bluetooth ON/OFF control
- Open desktop applications
- System shutdown and restart commands
- Time and Wikipedia information
- Voice-based desktop automation

---

## Technologies Used

- Python
- SpeechRecognition
- Pyttsx3
- PyWhatKit
- Wikipedia API
- WebBrowser Module
- OS Module
- Subprocess
- Platform Module

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/nova-ai-assistant.git
```

### Navigate to the Project Folder

```bash
cd nova-ai-assistant
```

### Install Required Libraries

```bash
pip install -r requirements.txt
```

---

## Required Python Libraries

```txt
pyttsx3
SpeechRecognition
wikipedia
pywhatkit
pyaudio
```

## Run the Project

```bash
python nova.py
```
